Explorando CSS - Principais Comandos

Seletores e Propriedades de Estilo

Os seletores de CSS são usados para selecionar os elementos HTML que você deseja estilizar. As propriedades de estilo determinam como esses elementos são exibidos.

p { color: #ff0000; } .classe { font-size: 16px; } #id { background-color: #f0f0f0; }

Layout e Posicionamento

O layout e o posicionamento no CSS controlam como os elementos são organizados na página.

display: block; float: left; position: relative;

Box Model

O Box Model define o espaço ao redor e dentro dos elementos.

padding: 10px; margin: 10px; width: 200px; height: 100px;

Tipografia

Estiliza o texto dentro dos elementos HTML.

font-family: Arial, sans-serif; text-align: center; text-decoration: underline;

Efeitos e Animações

Adiciona movimento e interatividade aos elementos.

transition: all 0.3s ease; animation: nomeAnimacao 1s infinite;
